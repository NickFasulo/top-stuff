# Top Stuff

Top Stuff is a web app that shows you previews for the top twelve video games and movies using react player.
* https://nickfasulo-top-stuff.herokuapp.com/

## Dependencies

```javascript
"dependencies": {
    "@testing-library/jest-dom": "^5.11.4",
```

## Built With

* [React](https://reactjs.org/) - Framework used.
* [React Player](https://www.npmjs.com/package/react-player) - Used to play video clips.
* [NPM](https://www.npmjs.com/) - Dependency management.
* [RAWG](https://rawg.io/apidocs) - API used to populate video game data.
* [TMDB](https://www.themoviedb.org/documentation/api) - API used to populate movie data.
